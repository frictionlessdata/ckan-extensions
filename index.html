CKAN Extensions

Introduction

CKAN is an open-source DMS (data management system) for powering data hubs and data portals. CKAN makes it easy to publish, share and use data. It powers catalog.data.gov, open.canada.ca/data, data.humdata.org among many other sites. We have collected and sorted by the stars count all the CKAN extensions available as repositories on Github and having at lease 1 star. Read more about CKAN. If you are interested in contributing to this livemark please follow this guide.
